# Quasar Enterprise AI Delivery Swarm

Band-powered multi-agent platform for regulated financial AI workflows.

## Current Status

Phase 1 Complete

### Features

- FastAPI backend
- Streamlit dashboard
- TimescaleDB
- Docker Compose
- Mock MCX Intelligence
- Mock Forex Intelligence
- Agent Swarm Monitor
- Log Viewer
- DB-backed candle and SMC label APIs
